# RPG de Terminal em Java 🎮

Um RPG simples feito em **Java** que roda no terminal.
O jogador enfrenta inimigos, ganha experiência e evolui durante as batalhas.

Este projeto foi criado com o objetivo de **praticar Programação Orientada a Objetos (POO)** e organização de projetos Java.

---

## 🚀 Funcionalidades

* Sistema de personagens
* Jogador com nível e experiência
* Inimigos com atributos próprios
* Sistema de batalha em turnos
* Cálculo de dano entre jogador e inimigo
* Evolução de nível do jogador

## ⚔️ Como funciona o combate

Quando o jogador encontra um inimigo, o jogo entra em um sistema de batalha baseado em turnos.

Exemplo:

```
Um Goblin apareceu!

Vida do Jogador: 100
Vida do Inimigo: 30

1 - Atacar
2 - Fugir
```

O jogador pode escolher atacar ou fugir.
O dano é calculado com base nos atributos de **ataque e defesa**.
